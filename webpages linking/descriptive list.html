Description of Courses

Data Analytics
This course covers the basics of data analytics,including data collection,data cleaning and basic statistical techniques.
Cyber Security
This course provides an overview of cyber security principles including basic concepts like threat types,risk management and fundamental protection.
Machine Learning
This course provides a foundational understanding of machine learning concepts,including supervised and unsupervised learning,basic algorithms.
Power BI
This course covers the basics of Power BI, including how to connect to data sources,create dashboards and generate reports.
Digital Marketing
This course provides a broad overview of digital marketing fundamentals,including search engine optimization(SEO),content marketing,social media marketing.
